\# Employee Management System (Hibernate + SQLite)



Desktop application built using Java Swing, Hibernate ORM, and SQLite.



\## Features

\- Add Employee

\- View Employee

\- Update Employee

\- Delete Employee

\- Search Employee



\## Technologies Used

\- Java

\- Swing

\- Hibernate ORM

\- SQLite

\- Maven
